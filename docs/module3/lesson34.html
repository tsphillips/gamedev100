Lesson 3.4: Game Board

That Crazy tiles[][] Array
Don't get too hung up over the tiles[][] array. It may seem confusing now, but as you go through the course and see how it's used, things should become clearer.

Now things are going to get more complex. We need a way to keep track of a game board, and all the tiles on the board. We're calling the game board a grid. As an example, a checkers board uses a grid that is eight tiles wide by eight tiles high.

Here's the code that creates our grid of tiles:

////////////////////////
// Lesson 3.4 Game Board

var tiles = [];
for (var i=0; i < grid_width; i++) {
   tiles[i] = new Array(grid_height);
} // for i

Conceptually, the grid looks something like this:

We can set a tile with: tiles[1][2] = 4;

Or we can check to see what a tile is with:

if (tiles[1][2] == 6) {  // Notice the == to check equality!
  // The tile 1 over and 2 down is set to 6!
  // Do something interesting.
} // if
Crazy code!
The random_tile() function might look intimidating. Don't panic. When you see code that looks too difficult to understand, skim trough it and then continue on. Come back to the code later; things may look easier after you learn more.

Now that we have a grid of tiles, let's make a couple helper functions. One function generates a random number, and the other scrambles the whole board.

random_tile() will return a random number. The number is a valid tile number for the current level.

var min_tile;
var max_tile;

function random_tile() {
   var tile;
   min_tile = ((level - 1) % number_of_tile_types) + 1;
   max_tile = min_tile + (tile_types_per_level - 1);
   tile = Math.floor(Math.random() * (max_tile - min_tile + 1)) + min_tile;
   if (tile > number_of_tile_types) {
       tile = tile - number_of_tile_types;
   } // if
   return tile;
} // random_tile()
Not Quite Random
The random_tile() function above technically does not return a "random" tile. Instead, it will return one of six tiles -- the six tiles appropriate for the level. That means we can call scramble_tiles() at the start of each level.

scramble_tiles() will scramble the whole game board and give us a whole bunch of randomly assigned tiles.


function scramble_tiles() {
   for (x=0; x < grid_width; x++) {
       for (y=0; y < grid_height; y++) {
           tiles[x][y] = random_tile();
       } // for y
   }  // for x
} // scramble_tiles()
scramble_tiles();  // Start the game with a grid of random tiles.

Later on we will use scramble_tiles() to reset the game board after a new level.
