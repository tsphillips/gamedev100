Lesson 6.2: Replacing Tiles

The first rule is that blank spots at the top of the grid need to be filled in with random tiles. We are only looking at the top row -- tiles[0][0], tiles[1][0], tiles[2][0], etc. At the same time we're also going to check to see if the player has gotten to the next level.

To check this rule we will use a for loop to check each tile in turn. If we find a blank spot (i.e., tiles[x][0] == 0), then we will set that tile to be a random tile. Way back in module three we created a random_tile() function that will give us a random tile.

Here's what a basic check of the top row looks like:

for (x = 0; x < grid_width; x++) {
    if (tiles[x][0] == 0) {
        tiles[x][0] = random_tile();
    } // if
} // for x

Let's modify the code just a little and put it all in a function:

function replace_tiles() {
   var created_tile = false;
   for (x = 0; x < grid_width; x++) {
       if (tiles[x][0] == 0) {
           tiles[x][0] = random_tile();
           created_tile = true;
       } // if
   } // for x
   if (created_tile == true) {
       check_new_level();
   } // if
} // replace_tiles()

Notice how we added a created_tile variable and can call a check_new_level() function? Any time we have to fill in any tiles on the top row, we will use that as an opportunity to check for a level-up. We could check for a level-up in other places, but it looks the best if the player levels up as the top row is filling in blank tiles.

Here's the level-check function:

function check_new_level() {
   if (swaps_made > swaps_per_level) {
       level++;
       swaps_made = 0;
       scramble_tiles();
   } // if
} // check_new_level()

The check is pretty simple. The swaps_made variable goes up by one for every swap the player makes. When the number of swaps made is more than the number of swaps needed per level (i.e., swaps_per_level), we:

  • raise the level by one (level++),
  • reset the swaps made counter (swaps_made = 0), and
  • completely reset the game board with new tiles (scramble_tiles()).

The scramble_tiles() function was introduced in module three.

Here's the final code for the replacing tiles rule:

/////////////////////////////
// Lesson 6.2 Replacing Tiles

// Replace tiles at the top
function replace_tiles() {
   var created_tile = false;
   for (x = 0; x < grid_width; x++) {
       if (tiles[x][0] == 0) {
           tiles[x][0] = random_tile();
           created_tile = true;
       } // if
   } // for x
   if (created_tile == true) {
       check_new_level();
   } // if
} // replace_tiles()

// Check for next level
function check_new_level() {
   if (swaps_made > swaps_per_level) {
       level++;
       swaps_made = 0;
       scramble_tiles();
   } // if
} // check_new_level()
